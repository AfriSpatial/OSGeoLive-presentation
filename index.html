Presented by Angelos Tzotsos / @tzotsos

What is OSGeoLive?

OSGeoLive is a GNU/Linux distribution containing more than 60 of the best open source geospatial applications

Oh hey, these are some notes. (hit 's' on your keyboard).

OSGeoLive is an OSGeo Project and is released as a self-contained Live DVD, USB thumb drive and Virtual Machine.

Open Source Geospatial Foundation (OSGeo)

Since 2006 A Non Profit Umbrella for:

  • GeoSpatial Free and Open Source Software
  • Education
  • Open Data

OSGeoLive contains:

  • 61 Open Source Geospatial Applications
  • Open Data (as part of application preconfiguration)
  • Project Overviews
  • Project Quickstarts (10 minute tutorials)
  • Translations
  • Development Tools and Libraries
  • Windows and Mac installers (available only on DVD)

Technology

  • Shell installers
  • Debian packaging / UbuntuGIS
  • based on Xubuntu LTS
  • built using the official Ubuntu spin-off tools
  • Continous Integration / nightly builds
  • Source Code Management in Subversion + Trac

What is new in 7.0?

  • Upgraded to Xubuntu 12.04.2 base
  • Upgraded all projects to their current stable version
  • New projects:
    • GeoNode
    • Iris
    • Leaflet
    • ncWMS
  • netCDF dataset
